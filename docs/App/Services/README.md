# Servis Katmanı

Uygulamanın iş mantığını (Business Logic) merkezi servis sınıfları yönetir.

*   **[AvailabilityService](./AvailabilityService/README.md)**: Ders, derslik ve gözetmen uygunluk sorgulama.
*   **[BaseService](./BaseService.md)**: Servisler için temel sınıf (Database, Logging, Transaction).
*   **[ScheduleService](./ScheduleService/README.md)**: Program kayıt ve düzenleme mantığı.
